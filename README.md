# js-demo

自己写的一些精选js效果类库，如瀑布流，360加速球等效果

## js瀑布流效果
封装的一个js类库,不依赖于第三方插件,功能有

* 初始化页面,将已有的dom以瀑布流形式展现
* 在已有的瀑布流后面添加新的元素
* 每次resize后js都会动态计算,以绝对布局absolute进行布局,适用于pc和移动端
* 便于拓展,有相应拓展示例
